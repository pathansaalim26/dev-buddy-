What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create the structure of web pages.

HTML is not a programming language. It does not perform logic or calculations. Instead, it tells the browser what content exists on a page and how that content is organized.

Using HTML, we define things like:

  • Headings
  • Paragraphs
  • Links
  • Images
  • Lists

Every website you see on the internet starts with HTML. CSS and JavaScript are added later to improve design and behavior.
